# 退職手当計算ツール（教育職）

北海道学校職員（教育職）の退職手当を正確に計算するWebアプリケーションです。

## 🌟 特徴

- ✅ **正確な計算**: 最新の退職手当制度に完全対応
- 📅 **自動元号表示**: 令和・平成・昭和を自動判定して表示
- 📈 **教職調整額の経過措置**: 令和8年〜令和13年の段階的引き上げに対応（4%→10%）
- 🎯 **勧奨退職の年度別対応**: 対象年齢の段階的引き上げに対応（51歳→55歳）
- 🖨️ **印刷対応**: そのまま印刷可能なレイアウト
- 📱 **レスポンシブデザイン**: PC・タブレット・スマホで利用可能

## 🚀 使い方

### オンライン版（推奨）
GitHub Pagesで公開している場合:
```
https://[あなたのGitHubユーザー名].github.io/退職手当計算アプリ/
```

### ローカル版
1. `index.html` をダウンロード
2. ブラウザで開く（ダブルクリック）
3. そのまま使用可能（インストール不要）

## 📝 入力項目

### 基本情報
- **採用年月日**: 正式採用日（臨時採用期間は除く）
- **退職予定年月日**: 退職する日付

### 給与情報
- **職務の級**: 1級〜4級、特2級
- **号俸**: 級に応じた号俸
- **職員区分（調整額）**: 第5号〜第11号
